# Covid19Prediction

A Cnn to predict Covid19 based on XRay Chest Images.

How it Works: 

first download the dataset from here https://github.com/ieee8023/covid-chestxray-dataset
place it in a folder and name it covid-chestxray-dataset then place it inside the data folder.

second download the dataset https://www.kaggle.com/datasets/praveengovi/coronahack-chest-xraydataset
place it ina  folder and name it covidDataset2 then place it inside the data folder.
